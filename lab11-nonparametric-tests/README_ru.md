# Лабораторная работа №11: Непараметрические тесты

[English version](README.md)

Применение критерия Манна–Уитни и критерия согласия χ² для анализа счетов в ресторане. Реализовано на Python с использованием `scipy`, `seaborn` и `matplotlib`.

## Ключевые результаты
- Счета за ужин статистически значимо выше, чем за обед (p < 0.001)
- Распределение счетов не соответствует экспоненциальному закону (χ²-тест, p ≈ 0)
- Визуализации подтверждают применимость непараметрических методов
