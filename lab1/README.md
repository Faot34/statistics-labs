# Lab 1 — Descriptive Statistics & Estimation

### Task 1 · Variant 3
Exploratory analysis of the Cars93 dataset. Horsepower distribution
described by car type using ECDF, histograms, and box-plots.

### Task 2 · Variant 3
MLE for the parameter θ of a Binomial(m, θ) distribution. Closed-form
derivation, bias/variance/MSE analysis, and simulation experiment at
m=4, θ=0.2 across sample sizes n=5 to n=1000.

### Task 3 · Variant 3
Confidence interval for the variance ratio σ₁²/σ₂² with unknown means,
based on the F-distribution. Theoretical derivation and coverage
experiment (1000 repetitions at n=25 and n=10000).

### Dependencies
Python 3.13 · pandas · numpy · matplotlib · seaborn · scipy
